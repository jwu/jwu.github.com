# MacVim Basic

## Keyboard Settings

Go To System Preferences → Keyboard → Keyboard, select “Use all F1,F2,….”

## Start From Terminal

```bash
cp mvim /usr/local/bin
```

## Install MacVim with other compile option

```bash
brew install macvim --with-cscope --with-lua --HEAD
```
